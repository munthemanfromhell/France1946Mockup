# French Social Democracy: The Fourth Republic

A native Dynamic Social Democracy election-viewer mod depicting the French legislative election of 10 November 1946 from the perspective of the SFIO.

## Run it

Upload `France1946Mockup_v1.json` to a public GitHub Pages repository, then enter the complete JSON URL in Dynamic Social Democracy's Mods screen.

Example:

`https://USERNAME.github.io/REPOSITORY/France1946Mockup_v1.json`

## Sequence

1. Election event
2. Pre-election briefing
3. National Assembly result, chamber and historical analysis
4. Government-formation choices
5. Aftermath text appended to the election page
6. Blank terminal page

The mod uses Dynamic's existing HTML, CSS, D3 parliament renderer and Dendry choice system. It does not contain a separate game engine.
